# System_Protocol_Zero_Entropy v1.1 [STATUS: BETA]

Системный протокол для подавления информационной энтропии и галлюцинаций в работе больших языковых моделей (LLM).

## О проекте
**System_Protocol_Zero_Entropy** — это логическая надстройка для нейросетей, которая отключает механизмы вежливости, «литературного сглаживания» и переводит модель в режим чистого аналитического вывода. 

## Компоненты
Репозиторий содержит два ключевых файла:
*   [`protocol_Zero_Entropy.yaml`](./protocol_Zero_Entropy.yaml) — технический промпт (инструкция) в формате YAML. Содержит архитектуру операционного режима и правила тегирования данных.
*   [`manual_Zero_Entropy.md`](./manual_Zero_Entropy.md) — подробное руководство пользователя: как внедрять протокол, как интерпретировать статусы.

## Быстрый запуск
1. Скопируйте содержимое файла `protocol_Zero_Entropy.yaml`.
2. Вставьте его в системное поле (System Prompt / Custom Instructions) вашей нейросети.
3. Ознакомьтесь с [Мануалом](./manual_Zero_Entropy.md) для понимания логики тегов `[CONFIRMED]`, `[STABLE]` и `[INFERENCE]`.
